##Stack

- [Next.js](https://nextjs.org/) with TypeScript

**Attention!** - there is a documented [issue](https://github.com/vercel/next.js/issues/42427) with Yarn so please use NPM


## Getting Started

First, run the development server:

```bash

# or
yarn dev
```
